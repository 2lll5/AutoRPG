# 零時迴圈 AutoRPG

一款會隨時間持續生長的繁體中文網頁 RPG。遊戲採用真正的故事圖（story graph）：每個選項都有自己的 `next` 節點，不會固定前往共同的下一幕。

## 故事結構

- 根節點的 3 個選擇會立刻進入 3 個完全不同的第二幕
- 每個故事節點固定提供 3 個選擇，各自指向不同節點
- 不同支線可以在後段交會，但不會清除玩家的旗標、物品與能力
- 交會節點會依照先前路徑顯示不同的「過往回聲」
- 核心故事內建 25 個節點與 6 個可直接抵達的不同結局
- 紅門、藍門、白門可選擇進入每小時持續生長的即時故事樹

## 每小時更新

GitHub Actions 每小時執行 `scripts/generate-story.mjs`：

- 每次同時新增破界、觀測、守名 3 個不同故事節點
- 每個新節點的 3 個選擇會分別指向下一小時的 3 個路線節點
- 玩家可以留在原路線，也可以在下一層交叉進入其他路線
- 網站直接讀取 GitHub 最新的 `public/story.generated.json`，不需等待 Vercel 重新部署
- 2026-07-31 晚間生成 3 個路線終幕與 9 個路線專屬最終結局，之後停止新增

## 存檔

Browser Cookie 保存：

- `currentNodeId`
- 經過的節點與選擇序列
- 勇氣、洞察、共鳴
- 特殊物品與分支旗標
- 重啟次數與彩蛋狀態

目前使用 `autorpg_state_v2`。舊版線性存檔因資料結構不相容，會從故事樹根節點重新開始。

## 本機執行

```bash
npm install
npm run dev
```

## 測試故事生成器

```bash
STORY_NOW=2026-07-15T09:00:00+08:00 npm run story:generate
```

同一小時重複執行不會建立重複節點。

## 部署

專案使用 Next.js，可直接部署至 Vercel。正式網站部署後，每小時內容由 GitHub Actions 更新；玩家重新整理頁面即可沿原本的節點路徑繼續。
